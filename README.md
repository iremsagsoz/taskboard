#  TaskBoard

React + Vite + Tailwind ile geliştirilmiş basit bir **CRUD Task/Todo uygulaması**.  
Görev ekleme, listeleme, güncelleme ve silme işlemleri yapılır.  
Veriler tarayıcıda **localStorage** üzerinde saklanır.

##  Özellikler
-  Görev Ekleme
-  Listeleme
-  Güncelleme (durum / içerik)
-  Silme
-  Arama / filtreleme (başlık, açıklama, durum)
-  localStorage ile kalıcı kayıt

##  Kullanılan Teknolojiler
- React (TypeScript)
- Vite
- Tailwind CSS
- localStorage
